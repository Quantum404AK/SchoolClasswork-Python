# SchoolClasswork-Python

Python is taught in my school as a programming language (6th Subject).
This is a repository for all the practicals performed at our computer lab.

1) MyDetails: A very basic script that prints down my personal details
2) VariableSwap: Values are asigned to variables 'a' and 'b'. The values are then swapped and printed.
3) Rectangle(AreaPerimeter): Input the length and breadth of a rectangle to get its area and perimeter.
4) Square(AreaPerimter): Input the length of a side of a square to get its area and perimeter.
5) ParityChecker: Checks if a number is even or odd
6) VotingEligibilty: Checks if a person is eleigible to vote in National elections
7) CheckingSignOfNumber: Checks if a number is positive, negative or zero.
8) GreatestFromTwoNumbers: Input two numbers and find which one is greater.
9) GreatestFromThreeNumbers: Input three numbers and find which one is the greatest.
10) LoopFirstTenNumbers: Prints the first ten natural nubers using "while..." loop
11) EvenAndOddNumbersInRange: Enter two numbers and find each the number of odd number and even numbers in the range
12) MultiplicationTable: Input a number and get its Multiplication Table.
13) AscendingAndDescendingTable: Input three numbers and get them arranged in Ascending and Descending order respectively.
